How to make a release
=====================
1. Make changes, test them
2. Edit qhexedit.h, version number
3. Edit pyqt5-pyproject.toml, version number
4. Edit pyqt6-pyproject.toml, version number
5. Edit changelog.md
6. Make doxygen documentation and check it
7. Commit changes
8. Create version tag

How to update docs
==================
1. Copy doxygen html to ...github.io
2. $ git add *
3. $ git commit -m "v0.x.y"
4. $ git push origin master
5. Create new release on github webpage
